# Vizualizátory BVS

Krátký repozitář obsahující interaktivní vizualizace:

- `index.html` — jednoduchý vstupní soubor s odkazy
- `abtree.html` — (a,b)-strom vizualizátor
- `rbtree.html` — červeno‑černý strom vizualizátor
